
## AI Wallpaper project


This is a fullstack web development project, you can visit the demo online [`AI Wallpaper Generator`](https://mywallpaper.vercel.app).

## tech stack

- Framework: Next.js
- UI: Tailwindcss
- AI: OpenAI
- database : supabase/PostgreSQL
- Hosting: AWS
- Auth: clerk
- Deployment: Vercel
